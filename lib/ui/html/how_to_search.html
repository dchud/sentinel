How to Search the Canary Database

Presently, the database search interface uses a general keyword search. This interface will match records where keyword terms are found in any of the title, authors, or abstracts of curated studies.

We are currently developing the database model and indexing strategies for the Canary Database. Because the studies we curate will include information from a variety of abstracting and indexing sources, we are exploring ways to make advanced searching capabilities easy to use across these source records. We expect to offer a flexible advanced search feature soon.

You can also browse the database by year, journal, or by methodology.
